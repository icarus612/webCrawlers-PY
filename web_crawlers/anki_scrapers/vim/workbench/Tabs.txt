<b>Vim Command:</b> :tabnew or :tabnew {page.words.file} | <b>Vim command</b> used to open a file in a new tab. 
<b>Vim Command:</b> Ctrl + wT | <b>Vim command</b> used to move the current split window into its own tab. 
<b>Vim Command:</b> gt or :tabn[ext] | <b>Vim command</b> used to move to the next tab. 
<b>Vim Command:</b> gT or :tabp[revious] | <b>Vim command</b> used to move to the previous tab. 
<b>Vim Command:</b> #gt | <b>Vim command</b> used to move to tab number #. 
<b>Vim Command:</b> :tabm[ove] # | <b>Vim command</b> used to move current tab to the #th position (indexed from 0). 
<b>Vim Command:</b> :tabc[lose] | <b>Vim command</b> used to close the current tab and all its windows. 
<b>Vim Command:</b> :tabo[nly] | <b>Vim command</b> used to close all tabs except for the current one. 
<b>Vim Command:</b> :tabdo command | <b>Vim command</b> used to run the command on all tabs (e.g. :tabdo q.
