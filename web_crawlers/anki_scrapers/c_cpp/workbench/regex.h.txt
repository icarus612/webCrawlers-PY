<b>Regex Function:</b> int regcomp(regex_t <i>*preg</i>, const char <i>*pattern</i>, int <i>cflags</i>); | <b>Regex function</b> used to compiles the source regular expression pointed to by pattern into an executable version and stores it in the location pointed to by preg. 
<b>Regex Function:</b> size_t regerror(int <i>errcode</i>, const regex_t <i>*preg</i>, char <i>*errbuf</i>,  size_t <i>errbuf_size</i>); | <b>Regex function</b> used to finds the description for the error code errcode for the regular expression  preg. 
<b>Regex Function:</b> int regexec(const regex_t <i>*preg</i>, const char <i>*string</i>, size_t <i>nmatch</i>, regmatch_t <i>*pmatch</i>, int <i>eflags</i>); | <b>Regex function</b> used to compares the null-ended string string against the compiled regular expression preg to find a match between the two. 
<b>Regex Function:</b> void regfree(regex_t *preg); | <b>Regex function</b> used to frees any memory that was allocated by regcomp to implement the regular expression preg.
